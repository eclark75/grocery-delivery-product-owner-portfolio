# Product Backlog

## Grocery Delivery App Product Backlog

| ID | User Story | Priority | Story Points | Status |
|----|------------|----------|--------------|--------|
| US-001 | As a new customer, I want to create an account so that I can save my information and place grocery orders. | High | 5 | Ready |
| US-002 | As a customer, I want to log into my account so that I can access my saved information and previous orders. | High | 3 | Ready |
| US-003 | As a customer, I want to search for grocery items so that I can quickly find products I need. | High | 5 | Ready |
| US-004 | As a customer, I want to add items to my shopping cart so that I can purchase multiple groceries at once. | High | 3 | Ready |
| US-005 | As a customer, I want to complete checkout securely so that I can receive my groceries. | High | 8 | Ready |

---

## Backlog Prioritization

1. User Registration
2. User Login
3. Product Search
4. Shopping Cart
5. Secure Checkout

---

## Sprint Goal

Deliver a Minimum Viable Product (MVP) that allows customers to register, browse groceries, add items to a cart, and complete a purchase.
