# Wireframes

## Grocery Delivery App Wireframes

The wireframes for this project were created in Figma to demonstrate the primary customer journey through the Grocery Delivery App.

---

## Screens Included

### 1. Home Screen
- Product search bar
- Featured grocery categories
- Popular products

### 2. Categories Screen
- Browse products by category
- Quick filtering
- Product grid layout

### 3. Product Details Screen
- Product image
- Description
- Price
- Quantity selector
- Add to Cart button

### 4. Shopping Cart
- List of selected items
- Quantity adjustment
- Order subtotal
- Checkout button

### 5. Checkout
- Delivery address
- Payment information
- Order summary
- Place Order button

### 6. Order Confirmation
- Order number
- Estimated delivery time
- Order summary
- Return to Home button

---

## Design Principles

The wireframes were designed using the following Product Owner goals:

- Simple user experience
- Fast navigation
- Minimal checkout friction
- Mobile-first design
- Clear calls to action
